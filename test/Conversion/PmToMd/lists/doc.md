# Lists Testing

Here is a simple unordered list:

-   Item 1
-   Item 2
-   Item 3

And a simple unordered one:

1.  Item 1
2.  Item 2
3.  Item 3

## Nested Lists

### Unordered

-   Item 1
    -   Item 1.1
        -   Item 1.1.1
        -   Item 1.1.2
    -   Item 1.2
-   Item 2
-   Item 3
    -   Item 3.1

### Ordered

1.  Item 1
    1.  Item 1.1
        1.  Item 1.1.1
    2.  Item 1.2
        1.  Item 1.2.1
2.  Item 2
    1.  Item 2.1

### List Items with Block Children

-   Item 1
    -   Item 1.1

        A **paragraph** in item 1.1

        1.  Item 1.1.1
        2.  Item 1.1.2
        3.  Item 1.1.3 (with [a link](https://v2editor.com "a link"))

    -   Item 1.2

    -   Item 1.3

        > A blockquote in a list item
-   Item 2
